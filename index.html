Welcome to FI-UBER documentation!

FI-UBER is a project developed as an assignment for Taller de Programación II at Universidad de Buenos Aires. The goal was implementing an app that allows passengers to hail a ride and drivers to charge fares and get paid.

In these docs you can find a general overview of said project, descriptions of some technical decisions and recomendations regarding developer experience.

Check out the Overview section for further information.

Note

This project is under active development.
